# Gimbal Control App

Este é um aplicativo Android simples para controlar o gimbal M01 via Bluetooth BLE. Possui interface acessível, botões grandes e joystick virtual para controle de movimentos.

## Funcionalidades
- Ligar/desligar o gimbal (se suportado via BLE)
- Alternar entre modo horizontal/vertical
- Mover o gimbal (esquerda, direita, cima, baixo) com joystick virtual

## Como usar
1. Instale o APK no seu dispositivo Android.
2. Conecte ao gimbal M01 via Bluetooth BLE.
3. Use os botões e o joystick para controlar o gimbal.

## Observações
- Projeto para fins de teste/protótipo. Não é necessário publicar na Play Store.
- Layout simples e funcional.
